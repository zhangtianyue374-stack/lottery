# 抽奖系统

基于 Vue 3 + Node.js + MongoDB 的抽奖系统。

## 技术栈

- **前端**: Vue 3 + Vite + TailwindCSS + Pinia
- **后端**: Node.js + Express + MongoDB
- **部署**: Vercel + Railway

## 快速开始

### 后端

```bash
cd server
npm install
cp .env.example .env
# 编辑 .env 填入 MongoDB URI
npm start
```

### 前端

```bash
cd client
npm install
npm run dev
```
